Dungeon Elimination: Dryption

What is DE: Kryption?

DE: Kryption is a close-quarters arena shooter set in a dark dungeon. Players are unable to see eachother by default and must use projectile flares in order to reveal eachother's location.

DE: Kryption featured fully functional online PVP combat.

Design Process

DE: Kryption was a semester long project created for MI 230. While making a digital game wasn't required for this course, I decided to push myself to create an online multiplayer game instead of a board game.

This would prove to be a very interesting challenge. Though I was familiar with the concepts of multiplayer archetecture as well as computer networking, I hadn't had the opportunity to get hands-on experience with programming a multiplayer game.

DE:K's Features

  • Online connectivity (playtest only: unavalible in demo)
  • Health, Ammo, and Visibility UI
  • Short-range dashing ability
  • Projectile-based combat
  • Light-based stealth mechanics
  • Experimental shader system
